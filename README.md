# GCP-Infra Structure By Terraform


## Terraform : 

- GKE Module
  
  - Cluster
  - Node
  - Service Account

- Network Module
   
   - VPC 
   - Firewall
   - Subnets
   - Nat 
 
 - VM Module 
  
   - vm instance with run Script.sh in Metadata 
 
   - Service Account
   
 - Docker 
  
    - Build and push dockerfile of pyhton app in GCP regiistery
    
 - Script.sh
    
    - install kubectl
    
    - install gcloud 
    
    - install git 
    
    
    
## Python-App

- Kubernetes
    
    - Deployment to run the app
    
    - Service to expose the app
